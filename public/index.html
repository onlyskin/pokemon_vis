A force directed graph created with D3.js: visually mapping links between Pokémon based on the number of shared moves in their movesets.

Two Pokémon will have a link drawn between them if the number of moves they share is more than or equal to the threshold. For example, as Bulbasaur, Ivysaur and Venusaur all share the same 14 moves in their moveset, if you set the threshold to 14 or less, they will have links drawn between them, but if the threshold is set any higher than 14 there will be no link drawn.

The higher you set the threshold, the fewer links will be drawn. This is because fewer Pokémon share so many of their moves with other Pokémon.

You can adjust the threshold for the number of shared moves needed using the input box below.

You can also drag Pokémon around using the mouse.
